Login Guide using Isomnia/Postman
Request formatting:
    - Method: POST
    - URL: http://localhost:3000/auth/login
    - Body JSON: 
        {
        "username": "admin",
        "password": "admin"
        }

On success should return
    200 OK:
    {
        "message": "Logged in to Grafana successfully"
    }



Get User Data
Request formatting:
    - Method: Get
    - URL: http://localhost:3000/grafana/user-data

On success should return
